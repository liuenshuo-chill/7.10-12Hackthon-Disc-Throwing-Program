# 飞盘夹爪适配改造

为了让夹爪能够夹住飞盘，我们从官方开源的模型仓库中下载了原有夹爪的 STL 模型，并进行了重新建模，使其轮廓更贴合飞盘的构造。

- 官方模型来源：[HighTorque-Robotics/Panthera-HT_Model](https://github.com/HighTorque-Robotics/Panthera-HT_Model/tree/main)

> **⚠️ 3D 打印注意**  
> 官方版本的 M2.5 螺丝接口边长为 `2.916mm`，实际打印时由于材料收缩及公差影响，**推荐将边长设置为 `3.300mm`** 以获得更好的配合效果。
